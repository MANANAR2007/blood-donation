# 🩸 Community Blood Donor Finder

---

## 🎯 Problem Context

Hospitals and patients often struggle to find blood donors quickly.  
The goal of this project was to build a React application that allows users to:

- View donors
- Filter by blood group
- Search by city
- Request help from available donors

---

## ✨ Features

- Fetch donor data using `useEffect`
- Filter donors by blood group
- Search donors by city (bonus feature)
- Sort donors by availability (bonus feature)
- Request Help button with state change
- Display total available donors count
- Loading spinner while fetching data
- No donors found UI state
- Responsive UI using Tailwind CSS

---

## 🛠️ Tech Stack

- React (Functional Components + Hooks)
- Vite
- Tailwind CSS
- JavaScript (ES6)
- JSONPlaceholder API (mock data)
